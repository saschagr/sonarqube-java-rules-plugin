This rule detects non documented getter (is/get) and setters

Noncompliant Code Example

public String getFoo() {
   return this.foo;
}

public void setFoo(String foo) {
   this.foo = foo;
}

Compliant Solution

/**
 * Get the value of foo.
 * @return the foo value.
 */
public String getFoo() {
   return this.foo;
}

/**
 * Set the value of foo.
 * @param foo the foo value to set.
 */
public void setFoo(String foo) {
   this.foo = foo;
}
